#  TWRP Device Tree for Poco X5 5G (moonstone) and Redmi Note 12 5G (sunstone)

The Poco X5 5G (codenamed "moonstone") is a Mid-range smartphone from Xiaomi. It was announced released in 2023, February 07. Also there is alternate version for this device, named Redmi Note 12 5G (codenamed "sunstone"). It was announced and released before X5 in 2022, October 27.
